## 🚀 Getting Started

1. **Edit Contact Information**
   - Update phone number, email, and location in the Contact section
   - Replace placeholder contact details with your actual information

2. **Customize Social Links**
   - Update social media links in the footer (Facebook, Twitter, LinkedIn, WhatsApp)

3. **Add Your Details**
   - Modify the hero section text to match your branding
   - Update the about section with your company information

4. **Enable GitHub Pages**
   - Go to repository settings
   - Enable GitHub Pages with main branch as source
   - Your site will be live at: `https://bhanuprakashchII67.github.io`

## 🎯 Customization Tips

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2c3e50;      /* Main color */
    --secondary-color: #3498db;    /* Accent color */
    --accent-color: #e74c3c;       /* Highlight color */
}
```

### Update Service Descriptions
Edit the service cards in `index.html` to add or modify your services.

### Add Your Logo
Add your business logo by replacing the navbar brand text with an image tag.
